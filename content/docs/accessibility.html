    All interactive components are keyboard accessible:

    Standard Keyboard Controls

    • Tab - Focus next element
    • Shift + Tab - Focus previous element
    • Enter - Activate button/select option
    • Space - Toggle checkbox/radio, activate button
    • Arrow Keys - Navigate menus, tabs, radio groups
    • Esc - Close modal/dropdown

    Color Contrast

    Frutjam's default color combinations meet WCAG AA standards. Never rely on color alone to convey information.

    Contrast Guidelines

    • Text and background: minimum 4.5:1 contrast ratio
    • Large text (18pt+): minimum 3:1 contrast ratio
    • Graphics and UI: minimum 3:1 contrast ratio
    • Use icons + text, not just color, for status

    Testing Accessibility

    Use these tools to test your implementation:

    • WAVE Browser Extension - Visual feedback on accessibility issues
    • axe DevTools - Automated accessibility testing
    • Lighthouse - Built into Chrome DevTools
    • Screen Readers - NVDA (Windows), JAWS, VoiceOver (Mac)
    • Keyboard Testing - Navigate with Tab only, no mouse

    Accessibility Checklist

    • ☐ All interactive elements are keyboard accessible
    • ☐ Focus indicators are visible
    • ☐ Color contrast meets WCAG AA standards
    • ☐ Form inputs have associated labels
    • ☐ Images have alt text
    • ☐ Modals trap focus and are announced
    • ☐ Status messages use aria-live regions
    • ☐ Semantic HTML elements are used
    • ☐ Language attribute is set on html element
    • ☐ Tested with keyboard and screen reader
