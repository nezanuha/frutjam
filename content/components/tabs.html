Tab components organize content into separate sections that users can switch between without leaving the page. Built with semantic HTML and radio input patterns, tabs provide intuitive navigation for grouping related information. The Frutjam tabs system supports multiple sizes, colors, and activation methods—CSS-only or JavaScript-controlled—making it ideal for dashboards, documentation, and tabbed interfaces.
