Frutjam themes use CSS custom properties to define colors, spacing, and visual style. Each theme is a collection of variables that control how components appear. Switch between light, dark, or fruit-named custom themes without writing any component code.

Theme Switching

Switch themes dynamically by setting the data-theme attribute:

Persist theme preference with localStorage:

function switchTheme(name) { document.documentElement.setAttribute('data-theme', name); localStorage.setItem('frutjam-theme', name); } // Load saved theme on page load const saved = localStorage.getItem('frutjam-theme') || 'light'; document.documentElement.setAttribute('data-theme', saved);

Creating a Custom Theme

Define new themes by setting CSS variables. Frutjam provides a set of color and spacing variables you can customize:

@layer theme { :is([data-theme="blueberry"]) { --color-base: oklch(98% 0.01 240); --color-on-base: oklch(15% 0.05 240); --color-primary: oklch(55% 0.25 240); --color-secondary: oklch(70% 0.18 200); --color-accent: oklch(75% 0.2 280); } }

Theme Variables

Core variables every theme should define:

  • --color-base - Background color
  • --color-on-base - Text color on base
  • --color-primary - Primary action color
  • --color-secondary - Secondary action color
  • --color-accent - Accent highlight color
  • --color-success - Success feedback (green)
  • --color-warning - Warning feedback (yellow)
  • --color-error - Error feedback (red)
  • --color-info - Neutral feedback (blue)

Naming Convention

Frutjam themes use fruit and jam-inspired names for custom themes:

  • blueberry - Cool blue tones
  • strawberry - Warm red tones
  • honeydew - Fresh green tones
  • apricot - Warm orange tones
  • blackberry - Deep purple tones
  • raspberry - Vibrant magenta tones

Best Practices

  • Define all required variables in each theme for consistency
  • Use OKLCH color space for perceptually uniform colors
  • Test color contrast with accessibility tools
  • Test light and dark themes with actual users
  • Save theme preference to localStorage
  • Respect system preference (prefers-color-scheme)
