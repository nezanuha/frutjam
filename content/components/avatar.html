Avatar components display user profile images or initials in a compact, circular or customizable container. Built with Frutjam's CSS custom properties, avatars represent users, contributors, or team members across your interface. The avatar system supports multiple sizes, colors, shapes, status indicators, rings, and group arrangements—ideal for user lists, comments, team displays, and profile sections.
