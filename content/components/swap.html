Swap components toggle between two content states or visual representations in response to user interaction. Built with semantic HTML and smooth transitions, swaps enable elegant state changes. The Frutjam swap system supports multiple styles and animation modes—ideal for theme toggles, mode switches, and interactive illustrations.

Basic Usage

Create a simple swap with text or emoji content that toggles on checkbox input.

Swap Styles

Apply animation styles to swaps for different visual effects when toggling.

Swap with Icons

Use SVG icons or other visual content that changes on toggle.

Swap with Colored Content

Display different colored boxes or content areas that swap on interaction.

Swap Buttons

Combine swap functionality with button styling for interactive toggles.

Swap Combinations

Mix different swap animation styles and content types.
