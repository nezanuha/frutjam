Accordion — Default
What is Frutjam?
Frutjam is a Tailwind CSS v4 component library built with CSS utilities and a PostCSS plugin. It ships pre-built components you can drop into any Tailwind project.
How does theming work?
All components are driven by CSS custom properties. Drop a data-theme attribute on any element to switch themes, or override tokens inline with style.
Is it accessible?
Yes — the accordion uses native <details> and <summary> elements, so keyboard navigation and screen readers work out of the box with zero JavaScript.
Can I use a prefix?
Absolutely. Pass prefix: "fj" in your @plugin config and every class becomes fj-accordion, fj-btn, etc.
Accordion — With Icons
Getting started
Install via npm: npm i frutjam, then add @import "frutjam/theme" and @plugin "frutjam" to your CSS.
Customising tokens
Override any token inside a @layer theme block or with a data-theme selector. Changes cascade automatically to every component.
