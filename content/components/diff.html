Diff components display differences between two versions of content, ideal for showing code changes, document revisions, or comparison views. Built with semantic HTML and color-coded styling, diffs help users understand what changed. The Frutjam diff system supports multiple types and styles—perfect for version control interfaces, revision history, and change tracking.

Class Type Description
diffBaseSide-by-side comparison container with a draggable divider
diff-item-1ModifierLeft (before) pane of the comparison
diff-item-2ModifierRight (after) pane of the comparison
diff-resizerModifierDraggable handle to resize the two panes
