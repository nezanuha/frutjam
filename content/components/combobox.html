Combobox components combine text input with dropdown selection, allowing users to either type custom values or choose from a curated list. Built with native input elements and flexible styling, comboboxes support autocomplete and filtering. The Frutjam combobox system supports multiple sizes, colors, and states—ideal for searchable selections, command palettes, and intelligent inputs.
