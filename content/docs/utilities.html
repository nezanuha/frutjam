Utility Best Practices

  • Use semantic component classes for complex components
  • Use utilities for layout and minor styling adjustments
  • Combine responsive prefixes for mobile-first design
  • Extract repeated utility combinations into custom classes
  • Use spacing scale (p-4, gap-6) consistently
  • Leverage flexbox and grid for layouts
  • Use color utilities from your theme
