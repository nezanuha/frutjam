Avatar components display user profile images or initials in a compact, circular or customizable container. Built with semantic HTML and flexible sizing, avatars represent users, contributors, or team members across your interface. The Frutjam avatar system supports multiple sizes, status indicators, rings, and group arrangements—ideal for user lists, comments, team displays, and profile sections.
